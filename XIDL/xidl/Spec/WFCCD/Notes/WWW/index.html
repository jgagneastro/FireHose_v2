The WFCCD IDL Pipeline

Brought to you by Jason X. Prochaska (UCO/Lick Observatory)

and Marla Geha (OCIW) and Mike Blanton (NYU)

The code is an IDL based suite of routines developed by JXP and modfied by MG and MRB. They are packaged together as part of the XIDL package maintained by JXP. They do require subroutines found within the SDSS packages. Information on installing the code and the code itself are in the links below.
