Machine builds:

These are tarballs that are fully built versions of the XIDL and also include the required SDSS packages. In principle, you should be able to unpack these codes and set the ENVIRONMENT variables and be ready to go. I expect the typical failure mode will be if you don't have the standard C libraries installed on your machine.

  • MacOSX 10.6; 64bit processor: TARBALL [240Mb; February 11, 2011]

Installing the code: The code is now distributed via SVN (or you can try to grab a machine-built tarball from above). These notes decribe what do to.

The current software includes data reduction packages for:
  • Includes HIRedux v2.3
  • Includes LowRedux (LRIS, DEIMOS, MagE, GMOS, etc.) v3.0
  • Includes ESIRedux v3.0
  • Includes MIKE redux v2.0
